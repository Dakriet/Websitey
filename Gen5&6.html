Generatie 5
In de Vijfde Generatie van Pokemon kwamen de spellen Pokemon Black & White uit in 2010. Het hoofdpersonage is Hilbert een jongen uit Nuvema Town in de Unova Regio, gebaseerd op de Verenigde Staten. Het avontuur van Hilbert begin in zijn eigen huis samen met Bianca en Cheren, zijn toekomstige rivalen, hij kan kiezen uit de Starter Pokemon Snivy, Tepig en Oshawott Ook Hilbert moet de 8 Pokemon Gyms verslaan, de eerste Gym is een ander Type per Starter Pokemon, als je Oshawott hebt gekozen, krijg je een Gras Type Gym, als je Tepig hebt gekozen, krijg je een Water Type Gym en als je Snivy hebt gekozen krijg je een Vuur Type Gym. de 7 andere Gyms zijn als volgt: Normaal Type -> Insect Type -> Elektrisch Type -> Grond Type -> Vlieg Type -> Ijs Type -> Draak Type. Daarna kan je naar de Pokemon League om de Elite Vier te verslaan, de Elite Vier van de Unova Regio hoef je niet in een bepaalde volgorde te verslaan. De Trainers zijn: Shauntal de Geest Type Trainer, Marshal de Vecht Type Trainer, Grimsley de Duister Type Trainer en Caitlin de Psychisch Type Trainer. Als je deze vier trainers weet te verslaan mag je naar de Kampioen, Alder, als je hem weet te verslaan mag je je Pokemon registreren in de Hall of Fame en is het spel klaar.
De remakes van de vijfde Generatie zijn anders, ze zijn een heel nieuw spel in dezelfde Regio genaamd, Pokemon Black 2 & White 2, het hoofdpersonage is nu Nate, zodra je bij de eerste Pokemon Gym komt is er al iets anders, de Leider van de Gym is Cheren, je rivaal in Black & White. In dit spel is hij de eerste Gym Leider de andere 7 Gyms zijn een beetje veranderd sinds de volgorde nu Normaal Type -> Gif Type -> Insect Type -> Elektrisch Type -> Grond Type -> Vlieg Type -> Draak Type -> Water Type is. De Elite Vier is hetzelfde maar de Kampioen is Iris, in Pokemon Black & White was zij leerling van de Draak Type Gym Leider Drayden, maar in dit spel is ze de Pokemon Kampioen. Nadat je haar verslaat mag je je Pokemon registreren in de Hall of Fame en is het spel klaar.
Generatie 6
In de Zesde Generatie van Pokemon bracht Nintendo de spellen Pokemon X & Y uit. Het hoofdpersonage is Calem uit Vaniville Town in de Kalos Regio, gebaseerd op Frankrijk. Hij krijgt zijn Starter Pokemon samen met Serena, Shauna, Tierno en Trevor. Hij kan kiezen uit Chespin, Fennekin en Froakie In deze generatie wordt er een nieuw Pokemon Type geÃ¯ntroduceerd: het Fee Type, Dit type is immuun voor het Draak Type, sterk tegen Duister, Vecht en Draak, maar zwak tegen Vuur Gif en Staal. Alhoewel hij samen met 4 andere mensen zijn Starter Pokemon krijgt blijft maar 1 trainer Calem zijn echte rivaal: Serena. Zij heeft meerdere Pokemongevechten met Calem door het spel heen. Calem moet langs de 8 Kalos Gyms. De Gyms zijn in de volgorde: Insect Type -> Steen Type -> Vecht Type -> Gras Type -> Elektrisch Type -> Fee Type -> Psychisch Type -> Ijs Type. Nadat hij klaar is met deze 8 Gyms, mag hij naar de Kalos League, om de Elite Vier uit te dagen. De Elite Vier bestaat uit Drasna de Draak Type Trainer, Wikstrom, de Staal Type Trainer, Malva de Vuur Type Trainer en Siebold de Water Type Trainer. Je hoeft ze niet in een bepaalde volgorde te verslaan. Nadat je de Elite Vier verslaat mag je Kampioen Diantha uitdagen als je klaar bent mag je naar de Hall of Fame om je Pokemon te registreren.
Generatie 6 had Remakes van de derde Generatie, genaamd OmegaRuby en AlphaSapphire. Deze spellen bieden niet veel nieuws alhoewel de Graphics erg mooi waren.
