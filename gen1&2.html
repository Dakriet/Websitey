Generatie 1
De eerste Generatie van Pokemon bestaat uit de spellen Pokemon Red, Blue, Green & Yellow. Deze spellen werden voor het eerst uitgebracht in Japan in 1996 (Pokemon Yellow werd iets later uitgebracht in 1998). Het hoofdpersonage is een stille jongen uit "Pallet Town", die later Red werd genoemd. In het spel gaat Red door de Kanto regio (gebaseerd op Kanto in Japan), waar zijn droom is om Pokemonmeester te worden. Hij krijgt zijn starter Pokemon van Proffesor Oak, samen met de Pokemon Trainer Blue, de kleinzoon van Proffesor Oak. Om Pokemonmeester te worden moet Red alle 8 Pokemon Gyms verslaan in deze volgorde: Steen Type -> Water Type -> Elektrisch Type -> Gras Type -> Gif Type -> Psychisch Type -> Vuur Type -> Grond Type. Nadat hij deze 8 Gyms verslagen heeft kan hij naar de Pokemon League waar hij de Elite Vier moet verslaan om daarna de Pokemon Kampioen te bereiken en hem ook te verslaan de Elite Vier bestaat uit: Lorelei: de Water & Ijs type Trainer, Bruno: de Steen & Vecht type trainer, Agatha: de Gif & Geest type Trainer en als laatste Lance: de Draak en Vlieg type Trainer. Als je alle vier de mensen heb verslagen komt je laatste uitdaging: de Kampioen. De Kampioen in dit spel is de jongen die je al vanaf het begin kent: Blue! Hij is samen met zijn starter Pokemon op het zelfde avontuur als jij gegaan en is nu de Pokemon Kampioen. Nadat je hem verslaat komt Proffesor Oak om je te feliciteren. Daarna moet je je Pokemon registreren in de Hall of Fame en is het spel klaar.
Generatie 2
De Tweede Geberatie van Pokemon bestaat uit Pokemon Gold, Silver en Crystal, spellen die in Japan in 1999 uitgebracht zijn en heeft ongeveer het zelfde verhaal. De Pokemon Trainer Gold wilt Pokemonmeester worden alleen dit keer in de Johto Regio, die gebaseerd is op de Chubu Regio in Japan. Hij haalt zijn starter Pokemon op bij een Proffesor, dit keer is het Proffesor Elm. Net nadat je het lab van Proffesor Elm verlaat kom je een mysterieuze jongen tegen. Hij heeft een Pokemon van Proffesor Elm gestolen. Zijn naam is Silver. Ook Gold moet 8 Gyms verslaan om bij zijn doel van Pokemonmeester te komen de 8 types van de Gyms in de Johto Regio zijn: Vlieg Type -> Insect Type -> Normaal Type -> Geest Type -> Vecht Type -> Staal Type -> Ijs Type -> Draak Type. Nadat je dit gedaan hebt moet je langs de Elite Vier. Deze Elite Vier bestaat uit: Will de Psychisch type Trainer, Koga de Gif type Trainer, Bruno de Vecht type Trainer (dit is dezelfde Bruno uit Pokemon Red, Blue & Yellow) en Karen de Duister type Trainer. Nadat je die 4 Trainers verslaat kom je bij de Kampioen: Lance, dezelfde Draak type Trainer als in de Eerste Generatie. Nadat je hem verslaat mag je je Pokemon registeren in de Hall of Fame en is het spel klaar. Dat zou je denken, maar dat is niet zo. Nadat je de Johto League verslaat kan je naar de Kanto Regio gaan en daar alle Gyms verslaan, dit keer zijn ze wel in een iets andere volgorde: Elektrisch Type -> Psychisch Type -> Water Type -> Gras Type -> Gif Type -> Steen Type -> Vuur Type -> Grond Type. Nadat je de 8 Gyms verslaat kan je naar je laatste uitdaging: Trainer Red. Na dit gevecht ben je officieel de Pokemonmeester en is het spel klaar.
